# Card Stats Tracker

Slay the Spire 2 模组 - 在单局游戏内追踪牌组中每张卡牌的统计数据。

## 功能

- 追踪每张卡牌的打出次数
- 追踪每张卡牌造成的伤害
- 追踪每张卡牌获得的格挡
- 追踪具有斩杀效果卡牌的斩杀次数
- 追踪每张卡牌施加的负面效果层数
- 追踪每张卡牌的抽牌数量
- 追踪每张卡牌获得的正面效果层数

## 安装

将 `CardStats.dll`、`CardStats.json` 和 `CardStats.pck` 放入游戏根目录的 `mods/CardStats` 文件夹：

```
Steam/steamapps/common/Slay the Spire 2/mods/CardStats/
```

## 使用

在**牌组界面**、**牌堆界面**和**历史界面**，左下角勾选「查看打出次数」，查看卡牌打出次数统计。

悬停卡牌时，tooltip 会显示该卡牌的详细统计数据：
- 打出次数
- 造成伤害（有伤害时显示）
- 获得格挡（有格挡时显示）
- 斩杀次数（有斩杀效果时显示）
- 施加负面效果层数（有负面效果时显示）
- 抽牌数量（有抽牌时显示）
- 获得正面效果层数（有正面效果时显示）

## 说明

战斗中保存并退出，统计数据回退到战斗开始时。
目前不兼容快速SL mod，使用快速SL不会回退次数统计，后续版本可能会支持。

## 编译

仅使用模组无需编译，直接下载 release 文件夹中的文件即可。

如需自行编译：
1. 安装 .NET SDK 9.0
2. 修改 `CardStats.csproj` 中的 `Sts2Dir` 为你的游戏安装路径
3. 运行 `dotnet build`
