# Card Stats Tracker

Slay the Spire 2 模组 - 追踪每张卡牌的打出次数。

## 安装

将 `CardStats.dll`、`CardStats.json` 和 `CardStats.pck`（如有）放入游戏 `mods` 文件夹：
```
Steam\steamapps\common\Slay the Spire 2\mods\CardStats\
```

## 使用

- **牌组界面** - 左下角勾选「查看打出次数」
- **牌堆界面** - 左下角勾选「查看打出次数」，抽牌堆/弃牌堆/消耗牌堆共用一个开关
- **历史界面** - 查看每局的卡牌使用统计

## 说明

- 同名卡牌独立计数（区分升级状态）
- 数据保存在本地，退出游戏不会丢失
- 战斗中保存退出会自动回退数据
